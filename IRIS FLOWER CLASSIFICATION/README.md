# Iris Flower Classification

This project demonstrates how to classify iris flowers into three species (setosa, versicolor, virginica) using machine learning in Python. The workflow includes data loading, exploration, visualization, preprocessing, model training, evaluation, and prediction.

## Dataset
- **File:** `IRIS.csv`
- **Columns:**
  - `sepal_length`: Sepal length in cm
  - `sepal_width`: Sepal width in cm
  - `petal_length`: Petal length in cm
  - `petal_width`: Petal width in cm
  - `species`: Iris species (Iris-setosa, Iris-versicolor, Iris-virginica)

## Steps
1. Import required libraries
2. Load and explore the dataset
3. Visualize feature relationships
4. Preprocess the data (encode labels, scale features)
5. Split data into training and test sets
6. Train a Random Forest classifier
7. Evaluate the model
8. Visualize confusion matrix
9. Predict on new data

## Usage
- Ensure `IRIS.csv` is in the project directory.
- Run `IRIS FLOWER CLASSIFICATION.py` to execute the workflow.
- View results in the terminal and generated plots.

## Requirements
- Python 3.x
- pandas, numpy, matplotlib, seaborn, scikit-learn
